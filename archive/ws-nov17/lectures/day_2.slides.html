Functions¶

Funtions we have used already¶

  • input(q): display message in q and return whatever user types
  • type(x): ask Python about the data type of x
  • len(x): ask Python to tell us the length of a variable x

Functions¶

  • a piece of code that we might want to run more than a few times

  • two things to remember: function name, input and output

  • our first function: Saying hello!

In [ ]:
def say_hello_to(name):
    greeting = "Hello " + name 
    return greeting
In [2]:
say_hello_to("everybody!")
Out[2]:
'Hello everybody!'

Function parts¶

def say_hello_to(name):
    greeting = "Hello " + name 
    return greeting
  • def: keyword (telling to Python that we are about to make a function now)
  • say_hello_to: function name, followed by parentheses (without spaces!)
  • name: function parameter
  • return: the stuff function returns to the place where we call it
  • say_hello_to(name): calling a function, which means using a function

Multiple arguments¶

In [8]:
def smaller(a, b):
    min_num = a
    if b < a:
        min_num = b
    return min_num
In [9]:

In [10]:

Exercise¶

In [15]:
def longer_string(stringA, stringB):
    if len(stringA) >= len(stringB):
        return stringA
    else:
        return stringB

There are countless ways to write a program. Your program is correct as long as it produces the correct output
