Functions¶

Yesterday, you all wrote your own programs to do things like calculating the total in a shopping cart or writing your own quiz. Today we introduce functions, which is a way to package the program up, so we don't need to write all the code again when we want to run our program again. In addition, if I want to share my program with others, function allows others to use my program without understanding the details of my code. Just like you don't have to understand all the mechanics inside a car to drive a car.

Funtions we have used already¶

  • input(q): display message in q and store whatever user types into a variable
  • type(x): ask Python about the data type of x
  • len(x): ask Python to tell us the length of a variable x
  • print(x): printing x to screen

Functions¶

  • a piece of code that we might want to run more than a few times
  • a nice way to package commands so we don't have to write them over and over again
  • our first function: Saying hello!

def smaller(a, b):

    return minimum

def longer_string(stringA, stringB):
    if len(stringA) >= len(stringB):
